TLDR - chmod
==========

Overview
--------
[chmod] change file mode bits 

Examples
--------

chmod -f
	supresses error emssages 
chmod +username, -username2
	+ gives permission to username
	- denies permission to username2

chmod -R
	change files and directories recursively
