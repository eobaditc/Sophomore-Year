TLDR - ls
==========

Overview
--------

[ls] list directory contents

Examples
--------

  ls -a 
     do not ignore entries starting with .

  ls -d
     list directories themselves, not contents 

  ls -C
     list entires by columns 

  ls -l 	
     long listing
