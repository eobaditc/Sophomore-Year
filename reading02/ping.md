TLDR - ping
=====

Overview
--------

send ICMP ECHO_REQUEST to network hosts

Examples
--------

- count, stop after sending a certain amount of requests

    $ping -c *count*

- ark, tag the packets outgoing 

    $ping -m

- umeric output only 

    $ping -n
